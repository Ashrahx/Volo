## About

**Volo** is a productivity tool focused on straightforward time and task management. It combines a structured task list with an integrated focus timer to help maintain a steady workflow. The application features a modular architecture and includes native internationalization, making it accessible in multiple languages directly out of the box.

---

## Tech Stack

- **Core:** Flutter
- **Language:** Dart
- **Storage:** Local Database
- **Routing:** Custom Shell Router
- **Typography:** Outfit Font
- **Internationalization:** Native multi-language support 

---

## Features

- **Task Management:** Create, organize, and track specific tasks.
- **Focus Timer:** Integrated timer system to structure work and break sessions.
- **Analytics & Reports:** Visual breakdown of task progress and focused time.
- **Global Localization:** Full translation support for 12+ languages (including English, Spanish, French, Japanese, Arabic, and more).
- **Customizable Settings:** App-wide preferences to adapt the experience to individual workflows.
